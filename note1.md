# markdown语法

### 1，标题

#+空格+文本（最多6个#)

------



### 2，字体

#### 1.字体加粗

```
**文本**
```

```
__文本__
```

#### 2.斜体

```
*文本*
```

```
_文本_
```

#### 3.粗斜体

```
***文本***
```

```
___文本___
```

#### 4.改颜色

```
<font color="">文本</font>
```

<font color="turquoise">turquoise</font>

<font color="cyan">cyan</font>

<font color="teal">teal</font>

#### 5.改大小

```
<font size=>文本</font>
```

<font size=3>size=3</font>

<font size=4>size=4</font>

#### 6.高亮

```
==文本==
```

==高亮文本==

------



### 3.线条

#### 1.分割线

使用3个以上的+/*/-来表示

#### 2.删除线

```
~~文本~~
```

#### 3.下划线

<u>需要使用html标签</u>

```
<u>文本</u>
```

------



### 4.列表

#### 1.无序列表

*/+/-space文本

- 1

- 2

- 3

  

#### 2.有序列表

数字+.+空格

1. 序列1

2. 序列2

3. 序列3

   

-----



### 5.链接

#### 1.访问外部链接

[链接名]（链接）

*注意用英文符号

[gdut](https://www.gdut.edu.cn/)

访问时需要按住Ctrl然后点击鼠标左键

#### 2.文件内跳转

[文本]（#标题）

[跳转到第一章](#1，标题)

----



### 6.插入图片

复制粘贴，右键选择缩放至合适大小
