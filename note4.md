# 面向对象

面向对象是一种编程思想，常见思维方式有面向过程和面向对象。

* 用面向过程形式了解去ATM机取钱

  ```
  1.带上银行卡
  2.排队
  3.把卡插入ATM机
  4.输入密码
  5.取钱
  ```

* 面向对象的形式

  寻找父母的帮助，让他们帮你去取钱

  ```
  优点：
  1.更符合人们的思考习惯
  2.把复杂的事情简单化
  3.把人们从执行者变成指挥者
  ```

  

## 面向对象三大特性

#### 封装

把属性和方法封装起来，仅对外提供方法让别人去访问，简化编程

日常生活中，手机，电脑都是封装

#### 继承

子类继承父类的属性和方法，使得子类对象（实例）具有父类的特征和行为，实现代码复用

只要是继承关系没那么都满足is-a关系



#### 多态

同一个函数，在不同场景下表现出不同形态，实现一象多用

同一个函数，接收不同的对象，有不同的效果

## 类和对象

#### 概念

##### 类：抽象的模板

###### 类变量

类变量属于类本身，而不是某个具体对象。所有这个类的对象共享一个类变量。修改类变量会影响到所有实例对象看到的值。

```python
class Student:
    school="GDUT"
s1=Student()
print(s1.school)
```



##### 对象：具体的实体

```
类->职员(抽象的概念)
属性：姓名，性别，年龄......
行为：行走，说话......

对象->张三(具体的实体)
姓名：张三，性别：男，年龄：32岁......
行走：爱走路上班，说话：很健谈，话很多....
```

```python
class Shape:
    def __init__(self, a, b):
        self.a = a
        self.b = b
```

* ——init——：构造函数，用于初始化对象属性
* self代表对象本身，用来访问对象的属性和方法
* self.a是长属性
* self.b是宽属性

##### 属性：对象的特征或状态

也叫实例变量

用来储存数据，在——init——方法中定义

##### 方法：对象的行为或功能

方法可以访问对象的属性，完成某些操作

eg.求三角形的面积，求面积就是一个“方法''

```python
class Shape:
    def __init__(self,a,b):
        self.a = a
        self.b = b
    def __str__(self):
        return f"三角形的面积是{self.a*self.b/2}"
triangle=Shape(5,6)
print(triangle)
```

###### ——str——的用处

控制对象以字符串的形式显示，一个类里只能有一个——str——方法，若有多个后面会覆盖前面的

###### 私有变量&私有方法

* 私有变量只能在类内部访问__用双下划线开头表示__
* 私有方法只能在类内部调用__用双下划线开头表示__

```python
class Student:
    def __init__(self,name,age):
        self.__name=name
        self.__age=age
    def __greet(self):
        print(f"hello,i'm {self.__name},i am {self.__age}years old")
    def say_hi(self):
        self.__greet()
s=Student("Cherie",19)
s.say_hi()
s.__greet()#报错
```
